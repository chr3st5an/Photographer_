Photographer

on Identity V. Its prefix is V, to get
started type Vhelp

like kick, mute, warn but also
has some fun commands such as fight
which allows you to fight against a friend

Between that the Bot also supports emotes and action commands such as kissing a friend

Identity V part. So, one of the main
